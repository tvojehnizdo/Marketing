## 🚀 Nasazení na Vercel

Chceš-li automatické nasazení a vlastní doménu, postupuj následovně:

### 1. Přihlas se do [vercel.com](https://vercel.com)
- Použij GitHub účet.

### 2. Vytvoř nový repozitář (nebo použij tento)

### 3. Importuj repozitář na Vercelu:
- Klikni **"Add New Project"**
- Zvol `tvojehnizdo/Marketing`
- Framework: **Other**
- Root Directory: **/** (nech prázdné)
- Klikni **Deploy**

### 4. (Volitelně) Přidej vlastní doménu `tvojehnizdo.cz`
- V nastavení projektu klikni „Domains“ → přidej doménu → nastav DNS podle návodu

---

## 📄 Obsah projektu

- `index.html` – hlavní stránka s výzvou k akci
- `Logo.png` – logo značky Tvoje Hnízdo
- `favicon.png` – ikona pro záložku
- `*.txt`, `*.docx` – podklady pro marketingovou kampaň
